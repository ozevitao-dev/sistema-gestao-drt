## PDF Export System (os-html-builder.ts)

### Layout Format
- **OS, Termo de Entrada, Venda (recibo), Atendimento Remoto (recibo)**: A4 landscape, 2 vias side-by-side, vertical dashed divider
- **Orçamento**: A4 portrait, single page
- **PNG exports**: 800px wide single card (no 2-vias)

### Configurações (PDF Customization)
- `usarMarcaDagua` (Boolean, default false): Logo as background watermark at 4% opacity
- `tamanhoLogo` (String, default "medio"): Logo size in header — pequeno/médio/grande
- Settings tab "Personalização do PDF" in Configurações page

### Header Format
- Logo on top (size per `tamanhoLogo`)
- Company info below in vertical column: nome, CNPJ, endereço, telefone, WhatsApp, email

### Signatures
- "Assinatura do Cliente" + "Assinatura do Técnico" in each via

### API Routes for Export
- OS: `/api/ordens/[id]/export` (PDF), `/api/ordens/[id]/export-png` (PNG)
- Orçamento: `/api/orcamentos/[id]/export` (PDF), `/api/orcamentos/[id]/export-png` (PNG)
- Venda: `/api/vendas/[id]/export` (PDF) — exports Lancamento with categoria='venda'
- Atendimento Remoto: `/api/atendimentos-remotos/[id]/export` (PDF)

### Key Files
- `lib/os-html-builder.ts`: All HTML builders (buildOSHtml, buildTermoEntradaHtml, buildVendaHtml, buildRemotoHtml, buildOrcamentoHtml + PNG variants)
- `lib/export-helpers.ts`: loadExportConfig(), generatePdf(), FullConfigData interface
